=== Woo SFTP Importer ===

Conecta-se a um servidor SFTP e importa produtos para o WooCommerce a partir de arquivos recebidos.

== Descrição ==

O Woo SFTP Importer é um plugin para WordPress que permite a conexão com servidores SFTP para importar produtos diretamente para o WooCommerce. Ele facilita a transferência de arquivos e a criação de produtos a partir desses arquivos, tornando o processo de gerenciamento de produtos mais eficiente.

== Instalação ==

1. Faça o download do plugin e extraia o conteúdo do arquivo ZIP.
2. Carregue a pasta `woo-sftp-importer` para o diretório `/wp-content/plugins/` do seu site WordPress.
3. Ative o plugin através do menu 'Plugins' no WordPress.
4. Configure as opções do plugin na área administrativa.

== Uso ==

Após a ativação, você pode acessar as configurações do Woo SFTP Importer no menu do WordPress. Insira as credenciais do servidor SFTP e configure as opções de importação de produtos.
